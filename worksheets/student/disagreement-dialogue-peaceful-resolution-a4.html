CIT-04 ใบงาน 1: CALM Dialogue Builder
CALM Dialogue Builder

ชื่อเล่น/รหัสผู้เรียน __________ วันที่ __________ (ไม่ต้องเขียนชื่อจริง / no real name required)

สถานการณ์สมมติ: สมาชิกสองคนต้องแบ่งอุปกรณ์ทำป้าย คนหนึ่งเริ่มใช้ก่อนโดยไม่ได้บอกอีกคน

1. แยกข้อมูล / Separate the information

ข้อเท็จจริงที่สังเกตได้
Observable fact		ความรู้สึกของผู้พูด
Speaker feeling		ความต้องการ
Need

2. เขียนประโยค CALM / Write a CALM statement

เมื่อฉันสังเกตว่า / When I notice…

ฉันรู้สึก / I feel… เพราะฉันต้องการ / because I need…

ฉันขอว่า / My request is…

3. ตรวจคำขอ / Check the request

☐ ชัดเจน ☐ ทำได้ ☐ ไม่ข่มขู่ ☐ อีกฝ่ายตอบหรือเสนอทางเลือกได้

4. ทางเลือกเมื่อยังไม่ตกลง / If agreement is not reached

เขียนสองทางเลือกที่เป็นธรรมและวันทบทวน / Write two fair options and a review time.

คำศัพท์: fact ข้อเท็จจริง · feeling ความรู้สึก · need ความต้องการ · request คำขอ
