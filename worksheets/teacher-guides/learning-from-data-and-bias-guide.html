คู่มือผู้สอน AI-03
Learning from Data and Bias

เวลา: 60–90 นาที · ช่วงชั้น: ป.4 · รูปแบบ: ไทย–อังกฤษ, static-first, local-only

สาระสำคัญ: แบบจำลองเรียนรูปแบบจากตัวอย่างที่มีป้ายกำกับ คุณภาพ ความหลากหลาย และความเหมาะสมของข้อมูลมีผลต่อข้อผิดพลาด แต่การเพิ่มข้อมูลเพียงอย่างเดียวไม่รับประกันความเป็นธรรม ต้องตรวจวัตถุประสงค์ ป้ายกำกับ ผลแยกกลุ่ม ข้อจำกัด การกำกับโดยคน และช่องทางแก้ไขด้วย

ผลลัพธ์การเรียนรู้

  • อธิบาย training data, label, model และ bias ด้วยภาษาของตนเอง
  • เปรียบเทียบผลจากชุดข้อมูลแคบกับชุดข้อมูลหลากหลาย
  • ใช้ FAIR: Find gaps, Add variety, Inspect errors, Report limits
  • แยกการคาดเดาของแบบจำลองออกจากข้อเท็จจริงและคุณค่าของบุคคล

แผน 60–90 นาที

  1. Engage 10 นาที: ถามว่า “ถ้าเราเห็นแอปเปิลสีแดงเท่านั้น เราจะจำแอปเปิลเขียวได้ไหม?” ห้ามใช้การเปรียบเทียบรูปลักษณ์หรือความสามารถของเพื่อน
  2. Explain 15 นาที: สาธิต example → label → pattern → prediction และย้ำว่าผลคือการคาดเดา
  3. Interactive 20 นาที: ทำ Fictional fruit trainer เปรียบเทียบชุด A/B ให้เด็กอ่านผลแยกชนิด ไม่ใช่คะแนนรวมอย่างเดียว
  4. Fairness lab 15 นาที: ใช้สถานการณ์ป้ายสองภาษาและ FAIR
  5. Worksheet/reflect 15–25 นาที: ทำใบงาน เลือกหลักฐาน และเขียนข้อจำกัด
  6. Exit 5 นาที: “A prediction is not a fact because…”

แนวคำตอบ

  • ลำดับใบงาน: รวบรวมตัวอย่าง → ให้ป้ายกำกับ → ฝึก/ใช้รูปแบบ → ทดสอบและตรวจข้อผิดพลาด
  • ช่องว่างของชุดแคบ: สี รูปทรง ขนาด และจำนวนของแต่ละหมวดไม่หลากหลายหรือสมดุล
  • หลักฐานที่ควรเลือก: คะแนนแยกกลุ่ม ตัวอย่างที่ทายผิด จำนวน/ความหลากหลาย และความถูกต้องของป้ายกำกับ
  • แผนที่ดี: เพิ่มตัวอย่างที่เกี่ยวข้องและหลากหลาย ตรวจป้าย ทดสอบแยกกลุ่ม เปิดเผยข้อจำกัด ให้คนรับผิดชอบตรวจ และมีช่องทางขอทบทวน

Safeguarding, privacy and fairness boundary

  • ใช้บัตรผลไม้ รูปร่าง หรือข้อความสมมติเท่านั้น ไม่ใช้ใบหน้า เสียง ชื่อ ผลการเรียน สุขภาพ ความพิการ เชื้อชาติ ศาสนา เพศ ที่อยู่ หรือลักษณะอ่อนไหวของเด็ก
  • ห้ามแบ่งหรือจัดอันดับผู้เรียนเป็น “ข้อมูลดี/ไม่ดี” “ฉลาด/ไม่ฉลาด” หรือให้แบบจำลองทำนายพฤติกรรม คุณค่า หรือโอกาสของคน
  • วิจารณ์ระบบและหลักฐาน ไม่วิจารณ์บุคคลหรือกลุ่มภาษา
  • อธิบายว่า balance ไม่ได้แปลว่าทุกกลุ่มต้องมีจำนวนเท่ากันเสมอ แต่ต้องเหมาะกับวัตถุประสงค์และผลกระทบ พร้อมทดสอบความต่างอย่างโปร่งใส
  • เว็บไซต์ไม่ส่งข้อมูลออกจากเครื่องและไม่ต้องสร้างบัญชี

การปรับสำหรับผู้เรียน / Differentiation

  • ให้ตอบด้วยคำพูด การเขียน ตาราง ภาพ หรือบัตรสัญลักษณ์
  • อ่านโจทย์ให้ฟังและใช้บัตรตัวอย่างที่มีข้อความกำกับ ไม่พึ่งสีอย่างเดียว
  • ลดจำนวนตัวอย่างแต่คงการเปรียบเทียบสองชุด
  • ผู้เรียนที่พร้อมสามารถออกแบบ test set ใหม่และอธิบายว่าทำไมไม่ควรใช้ข้อมูลฝึกเป็นข้อมูลทดสอบทั้งหมด

Rubric 4 ระดับ

ระดับความเข้าใจข้อมูลการตรวจความเป็นธรรมการสื่อสารข้อจำกัด
4อธิบาย example–label–model–prediction ได้ถูกต้องใช้ FAIR ครบและเลือกหลักฐานแยกกลุ่มเสนอ human review และช่องทางแก้ไขชัดเจน
3อธิบายแนวคิดหลักได้พบช่องว่างและเสนอเพิ่มความหลากหลายบอกข้อจำกัดได้อย่างน้อยหนึ่งข้อ
2ระบุคำศัพท์บางส่วนเห็นความต่างของผลแต่เหตุผลยังไม่ครบต้องมีคำถามนำ
1ยังสับสนว่าการคาดเดาคือข้อเท็จจริงดูคะแนนรวมอย่างเดียวยังไม่อธิบายข้อจำกัด
