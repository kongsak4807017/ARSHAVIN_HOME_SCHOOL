คู่มือผู้สอน MAKER-09
Heat, Insulation and Safe Container Design

เวลา / Duration: 60–90 นาที · ระดับ / Level: ป.4 · รูปแบบ: ไทย–อังกฤษ, static-first, local-only

ขอบเขตความปลอดภัย / Safety boundary: ค่าเริ่มต้นคือกิจกรรมบนหน้าจอหรือบัตรข้อมูล หากมีชิ้นงาน ใช้เฉพาะกระดาษ ผ้าสะอาด กล่องหรือถ้วยเปล่าที่อุณหภูมิห้อง น้ำหนักเบา และขอบไม่คม ผู้ใหญ่เป็นผู้ตัดหรือเจาะ ห้ามเปลวไฟ เทียน เตา น้ำเดือด ไอน้ำ ของร้อน ของเย็นจัด น้ำแข็งแห้ง สารเคมี เครื่องทำความร้อน การอุ่นอาหาร การทดลองกับผิวหนัง หรือการให้เด็กถือภาชนะที่อาจร้อน หากพบความร้อน รั่ว แตก กลิ่นผิดปกติ หรือความไม่มั่นคง ให้หยุด ถอยออก และแจ้งผู้ใหญ่

ผลลัพธ์การเรียนรู้ / Learning outcomes

  1. อธิบายความร้อนในฐานะการถ่ายโอนพลังงานจากบริเวณอุณหภูมิสูงกว่าไปต่ำกว่า
  2. แยกการนำ การพา และการแผ่รังสีในสถานการณ์ง่าย โดยยอมรับว่าอาจเกิดพร้อมกัน
  3. อธิบายว่าฉนวนช่วยชะลอ ไม่ได้หยุดการถ่ายโอนทั้งหมด
  4. สร้าง fair test ที่เปลี่ยนทีละหนึ่งตัวแปรและเขียนข้อสรุปไม่เกินหลักฐาน
  5. ออกแบบภาชนะจำลองที่มั่นคง เข้าถึงได้ และมี stop rule

แผนการสอน 60–90 นาที

เวลากิจกรรมหลักฐาน
0–10 นาทีถามว่าเหตุใดกล่องเก็บความเย็นหรือปลอกแก้วจึงมีหลายชั้น แยก “อุณหภูมิ” ออกจาก “ความร้อน” โดยไม่ใช้การสัมผัสของจริงคำอธิบายเบื้องต้น
10–25 นาทีสอน conduction–convection–radiation ด้วยภาพ ลูกศร และตัวอย่างสมมติ ย้ำว่าระบบจริงมีหลายเส้นทางพร้อมกันแผนผังคำศัพท์
25–40 นาทีทำกิจกรรม comparison บนเว็บ ผู้เรียนอ่านผลสมมติและเลือกข้อสรุปที่ไม่อ้างเกินหลักฐานcomparisonComplete
40–60 นาทีทำใบงานหน้า 1 และออกแบบ fair test หน้า 2 หากสร้างต้นแบบ ให้ใช้เฉพาะวัสดุอุณหภูมิห้องตารางและภาพร่าง
60–75 นาทีตรวจความมั่นคง จุดจับ ฝา การเข้าถึง และ stop rule เปรียบเทียบ trade-offsdesignComplete
75–90 นาทีแบบประเมิน สะท้อนคิด และ feedback รายบุคคลโดยไม่จัดอันดับquizComplete + exit ticket

แนวคำตอบ / Answer guidance

  • กิจกรรม 1: เลือก “ฉนวนและฝาปิดอาจช่วยชะลอ...” เพราะคะแนนเป็นข้อมูลสมมติและผลจริงขึ้นกับความหนา ช่องอากาศ ฝา เวลา และสภาพแวดล้อม
  • กิจกรรม 2: fair = one-variable; design = stable; stop = adult
  • Quiz: q1 slow, q2 contact, q3 room
  • ใบงาน: ข้อสรุปที่ดีใช้คำว่า “ภายใต้เงื่อนไขนี้” หรือ “อาจช่วยชะลอ” และระบุสิ่งที่ยังไม่ได้ตรวจ เช่น food safety, fire resistance หรือ product durability

ความเข้าใจผิดที่ควรแก้ / Misconceptions

ความเข้าใจผิดคำอธิบายแก้ไข
ฉนวนหยุดความร้อนทั้งหมดฉนวนช่วยลดอัตราการถ่ายโอนภายใต้เงื่อนไขหนึ่ง แต่การถ่ายโอนยังเกิดได้
วัตถุ “มีความร้อน” เท่ากับความร้อนเป็นสารอยู่ข้างในในบทนี้ heat หมายถึงพลังงานระหว่างการถ่ายโอนเพราะความต่างอุณหภูมิ
หนาที่สุดดีที่สุดเสมอต้องพิจารณาน้ำหนัก ความมั่นคง ความชื้น การใช้งาน ความปลอดภัย และข้อจำกัด
แตะด้วยมือคือเครื่องวัดที่ดีการรับรู้ของผิวหนังไม่ใช่การวัดที่ปลอดภัยหรือมาตรฐาน และห้ามใช้กับของร้อน
ผลจากวัสดุจำลองรับรองภาชนะอาหารกิจกรรมไม่ทดสอบการสัมผัสอาหาร ความทนไฟ สารเคมี หรือมาตรฐานผลิตภัณฑ์

การเข้าถึงและการมีส่วนร่วม / Accessibility

  • ใช้ select, radio และ button ที่ทำงานด้วยคีย์บอร์ด พร้อม fieldset, legend และ live feedback
  • ให้ตอบด้วยคำพูด ภาพ ลูกศร บัตรคำ การเขียนตามคำบอก หรือ AAC ได้
  • ผู้เรียนที่ไม่สัมผัสวัสดุทำกิจกรรมจากบัตรข้อมูลและภาพร่างทั้งหมดได้
  • ไม่มี timer, speed contest หรือ drag-only interaction
  • ไม่ใช้สีเป็นช่องทางเดียว ให้ติดป้ายข้อความกับลูกศรทุกเส้นทาง

Rubric 4 ระดับ

ระดับความเข้าใจวิทยาศาสตร์การออกแบบและหลักฐานความปลอดภัยและการสื่อสาร
4 ก้าวหน้า / Advancedอธิบาย 3 เส้นทางและข้อจำกัดได้ชัดfair test ครบ ทบทวนจากหลักฐาน และระบุ trade-offstop rule ชัด ใช้ภาษาสองภาษาและไม่อ้างเกินหลักฐาน
3 บรรลุ / Proficientอธิบายฉนวนและอย่างน้อย 2 เส้นทางได้เปลี่ยนหนึ่งตัวแปรและบันทึกผลครบระบุความเสี่ยงหลักและบทบาทผู้ใหญ่
2 กำลังพัฒนา / Developingรู้ว่าฉนวนชะลอแต่ยังสับสนเส้นทางแผนมีหลายตัวแปรหรือข้อจำกัดไม่ครบต้องเตือนเรื่อง stop rule หรือคำกล่าวเกินหลักฐาน
1 เริ่มต้น / Beginningยังคิดว่าฉนวนสร้างความร้อน/ความเย็นยังไม่มี fair test หรือหลักฐานยังเสนอของร้อนหรือการสัมผัสเป็นการทดสอบ

การรับมือข้อมูลสุขภาพหรือเหตุการณ์จริง

ไม่ถามเด็กเกี่ยวกับแผลไหม้ ประวัติการบาดเจ็บ สภาพบ้าน หรืออุปกรณ์ทำอาหาร หากเด็กเปิดเผยเหตุการณ์อันตราย ให้หยุดกิจกรรม รักษาความเป็นส่วนตัว และใช้กระบวนการคุ้มครองเด็ก/ปฐมพยาบาลของสถานศึกษาโดยผู้ใหญ่ที่รับผิดชอบ กิจกรรมนี้ไม่สอนการรักษาแผลไหม้
