Teacher guide · AI-09

คู่มือผู้สอน: ระบบรู้จำภาพ ข้อผิดพลาด และความเป็นส่วนตัว
Image Recognition, Errors and Privacy

เวลาแนะนำ 60–90 นาที ใช้บัตรข้อมูลสมมติและหน้าจอเท่านั้น ไม่เปิดกล้องและไม่เก็บภาพผู้เรียน

ผลลัพธ์การเรียนรู้ / Outcomes

  • อธิบาย feature → comparison → label ได้โดยไม่กล่าวว่า AI “มองและเข้าใจเหมือนคน”
  • จำแนก false positive, false negative และ unknown
  • ตรวจคุณภาพข้อมูลและข้อจำกัดก่อนใช้ผลลัพธ์
  • ปฏิเสธ face recognition, biometric identification และ child profiling ในกิจกรรม

ลำดับการสอน / Sequence

  1. 10 นาที: ทบทวน feature, label และเหตุผลที่คุณลักษณะเดียวอาจทำให้ผิด
  2. 20 นาที: ทำกิจกรรม feature-matching และบังคับใช้คำตอบ “ไม่แน่ใจ” เมื่อข้อมูลไม่พอ
  3. 15 นาที: วิเคราะห์ false positive และ false negative ด้วยกรณีของเล่นรูปใบไม้
  4. 20 นาที: ทำใบงานออกแบบระบบผลกระทบต่ำด้วย LENS
  5. 10–15 นาที: แบบประเมินและสะท้อน privacy/human-review rule

แนวคำตอบ / Answer guidance

  • ใบไม้จริง: หลายคุณลักษณะร่วมกันสนับสนุนป้าย leaf แต่ยังเป็นแบบจำลอง ไม่ใช่การรับรองทางพฤกษศาสตร์
  • ของเล่นรูปใบไม้: ถ้าระบบป้ายเป็นใบไม้จริง ถือเป็น false positive
  • ใบไม้จริงที่ระบบพลาด: false negative
  • ภาพไม่ชัด: คำตอบที่รับผิดชอบคือ unknown/abstain ตรวจข้อมูลเพิ่ม และให้มนุษย์ทบทวน
  • privacy: ใช้บัตรวัตถุสมมติ ไม่ใช้ใบหน้าเด็ก ชื่อ เสียง ตำแหน่ง สุขภาพ หรือข้อมูลชีวมิติ

ความเข้าใจผิดที่ควรแก้ / Misconceptions

  • “AI เห็นเหมือนมนุษย์” — ระบบคำนวณจากข้อมูลและคุณลักษณะที่ออกแบบหรือเรียนรู้
  • “มั่นใจสูงแปลว่าถูก” — confidence ไม่รับรอง correctness
  • “ภาพมากขึ้นย่อมดีเสมอ” — ต้องถามถึงความยินยอม ความจำเป็น คุณภาพ ความเป็นตัวแทน และสิทธิของเด็ก
  • “ข้อผิดพลาดทุกชนิดเหมือนกัน” — false positive และ false negative มีผลกระทบต่างกันตามบริบท
  • “ใบหน้าคือข้อมูลทั่วไป” — ใบหน้าอาจเป็นข้อมูลชีวมิติและต้องคุ้มครองเป็นพิเศษ

Safeguarding และ privacy boundary

ห้าม: เปิดกล้อง อัปโหลดภาพหรือเสียงเด็ก ขอรูปครอบครัว ทำ face recognition, biometric identification, emotion recognition, attendance tracking, behavioural scoring หรือจัดอันดับเด็ก ห้ามใช้ผลระบบตัดสินสิทธิ สุขภาพ ความปลอดภัย การลงโทษ หรือโอกาสทางการศึกษา

เมื่อผู้เรียนเปิดเผยว่ามีการถ่ายภาพ ติดตาม หรือระบุตัวโดยไม่ยินยอม ให้หยุดกิจกรรม รักษาความเป็นส่วนตัว และส่งต่อผู้ใหญ่ตามระบบคุ้มครองเด็กของสถานศึกษา

Accessibility และการมีส่วนร่วม / Access

  • ตอบได้ด้วยคำพูด ภาพ บัตรคำ ลูกศร การเขียนตามคำบอก หรือ AAC
  • อ่านทุกผลลัพธ์เป็นข้อความ ไม่ใช้สีเพียงอย่างเดียว
  • ไม่มี timer, reaction-time comparison หรือ drag-only interaction
  • ให้เวลาประมวลผลและอนุญาตให้ข้ามกิจกรรมที่เกี่ยวกับภาพได้โดยไม่มีผลเสีย

Rubric 4 ระดับ

ระดับหลักฐานการเรียนรู้
4 — ชัดเจนมากอธิบาย feature/label แยกข้อผิดพลาด ระบุ unknown ตรวจ privacy/fairness และสร้าง human-review rule ได้
3 — บรรลุจับคู่คุณลักษณะและข้อผิดพลาดส่วนใหญ่ถูก พร้อมปกป้องภาพเด็กและขอมนุษย์ทบทวน
2 — กำลังพัฒนาเข้าใจบางส่วนแต่ยังพึ่งคุณลักษณะเดียวหรือสับสน false positive/negative
1 — ต้องสนับสนุนยังเชื่อผลระบบโดยไม่ตรวจ หรือยังไม่เห็นความเสี่ยงด้านภาพและข้อมูลชีวมิติ
