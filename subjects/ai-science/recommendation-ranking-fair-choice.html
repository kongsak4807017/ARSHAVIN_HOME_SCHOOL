ระบบแนะนำ การจัดอันดับ และการเลือกอย่างเป็นธรรม
Recommendation, Ranking and Fair Choice

คำถามนำ / Big question: ระบบควรแนะนำทางเลือกอย่างไรโดยไม่ซ่อนเกณฑ์หรือควบคุมคน?

ไปที่ / Jump to: เรียนรู้ · ตัวแนะนำ · ความเป็นธรรม · แบบฝึกหัด · สำหรับผู้ใหญ่

เป้าหมาย / Learning goals

  • อธิบายความต่างระหว่างการแนะนำและการตัดสินแทน
  • อ่านเกณฑ์ น้ำหนัก และข้อจำกัดของการจัดอันดับ
  • เปลี่ยนเกณฑ์และดูว่าผลลัพธ์เปลี่ยนอย่างไร
  • รักษาทางเลือกของผู้ใช้และห้าม profiling หรือจัดอันดับเด็ก

คำศัพท์ / Vocabulary

recommendation — การแนะนำ
การเสนอทางเลือกที่อาจเหมาะ โดยผู้ใช้ยังตัดสินใจเอง
ranking — การจัดอันดับ
การเรียงรายการตามคะแนนหรือกฎที่กำหนด
criterion — เกณฑ์
สิ่งที่ใช้เปรียบเทียบ เช่น ระยะเวลา ความยาก หรือการเข้าถึง
weight — น้ำหนัก
ความสำคัญที่ให้แต่ละเกณฑ์
user control — การควบคุมโดยผู้ใช้
ผู้ใช้ดู เปลี่ยน ปิด หรือไม่ทำตามคำแนะนำได้

กรอบ CLEAR / CLEAR frame

  1. C — Clarify the goal: ระบุเป้าหมายจริง
  2. L — List criteria: เปิดเผยเกณฑ์ที่ใช้
  3. E — Explain weights: อธิบายน้ำหนักและข้อจำกัด
  4. A — Allow choice: ให้คนเปลี่ยนเกณฑ์ ดูทางเลือกอื่น และตัดสินใจเอง
  5. R — Review fairness: ตรวจว่าใครอาจถูกมองข้ามและเปิดทางทบทวน

ขอบเขต: ใช้รายการกิจกรรมสมมติเท่านั้น ไม่ใช้ชื่อ ภาพ เสียง คะแนน สุขภาพ พฤติกรรม ฐานะ ตำแหน่ง หรือข้อมูลครอบครัว และห้ามใช้ระบบนี้จัดอันดับหรือทำนายคุณค่าของเด็ก

แบบฝึกหัด / Quick check

1) การแนะนำที่ดีควรทำอะไร?
2) เมื่อเปลี่ยนน้ำหนักเกณฑ์ ผลอันดับอาจเป็นอย่างไร?
3) ระบบนี้ควรใช้จัดอันดับเด็กหรือไม่?

สรุป / Summary

อันดับไม่ใช่ความจริงสากล แต่เป็นผลจากเป้าหมาย เกณฑ์ ข้อมูล และน้ำหนัก ระบบที่รับผิดชอบต้องโปร่งใส ให้ผู้ใช้ควบคุม และไม่จัดอันดับเด็กหรือกำหนดโอกาสแทนมนุษย์
