รูปแบบ คุณลักษณะ และกรณีที่กฎใช้ไม่ได้
Patterns, Features and Edge Cases

คำถามนำ / Big question: กฎที่ดูเหมือนใช้ได้กับตัวอย่างฝึก จะยังใช้ได้กับของชิ้นใหม่ที่แตกต่างออกไปหรือไม่?

ไปที่ / Jump to: เรียนรู้ · สร้างกฎ · ทดสอบกรณีพิเศษ · แบบฝึกหัด · สำหรับผู้ใหญ่

เป้าหมาย / Learning goals

  • อธิบายความหมายของ pattern, feature, label, training example, test example
  • สร้างกฎจำแนกจากคุณลักษณะที่สังเกตได้ของสิ่งของสมมติ
  • แยกตัวอย่างฝึกออกจากตัวอย่างทดสอบ เพื่อดูว่ากฎใช้กับของใหม่ได้จริงหรือไม่
  • ค้นหา edge case — กรณีพิเศษ และปรับกฎโดยไม่กล่าวโทษคนหรือข้อมูล

คำศัพท์ / Vocabulary

pattern — รูปแบบ
สิ่งที่เกิดซ้ำหรือความสัมพันธ์ที่เราสังเกตพบ
feature — คุณลักษณะ
ข้อมูลที่ใช้พิจารณา เช่น วัสดุ รูปร่าง มีฝาปิดหรือไม่
training example — ตัวอย่างฝึก
ตัวอย่างที่ใช้สร้างหรือปรับกฎ
test example — ตัวอย่างทดสอบ
ตัวอย่างใหม่ที่ไม่ควรใช้สร้างกฎ เพื่อดูว่ากฎทำงานกับสิ่งที่ยังไม่เคยเห็นหรือไม่
edge case — กรณีพิเศษ
ตัวอย่างที่ต่างจากกรณีทั่วไปและอาจทำให้กฎผิดพลาด

หลักสำคัญ / Key idea

การจำแนกสิ่งของเริ่มจากเลือกคุณลักษณะที่เกี่ยวข้อง สังเกตรูปแบบจากตัวอย่างฝึก แล้วตรวจด้วยตัวอย่างทดสอบที่แยกไว้ หากเราแอบดูคำตอบของชุดทดสอบและแก้กฎตามทุกข้อ ผลทดสอบจะไม่น่าเชื่อถือ

  1. LOOK — มอง: ระบุคุณลักษณะที่วัดหรือสังเกตได้
  2. LEARN — เรียน: สร้างกฎจากตัวอย่างฝึก
  3. LOCK — แยก: เก็บตัวอย่างทดสอบไว้ ไม่ใช้สร้างกฎ
  4. TEST — ทดสอบ: ตรวจของใหม่และบันทึกทั้งถูกและผิด
  5. REVISE — ปรับ: อธิบายข้อจำกัด เพิ่มตัวอย่างที่เหมาะสม แล้วทดสอบใหม่ด้วยชุดใหม่

ขอบเขตบทเรียน: ใช้เฉพาะสิ่งของในบ้านแบบสมมติ ไม่ขอภาพ เสียง ชื่อ ตำแหน่ง สุขภาพ ผลการเรียน หรือลักษณะส่วนบุคคล และห้ามใช้กฎนี้จัดประเภท ประเมิน หรือตัดสินคน

กิจกรรม 1: นักออกแบบกฎจำแนก / Rule-builder lab

ตัวอย่างฝึกมี 4 ชิ้น: กล่องกระดาษมีฝาและขวดพลาสติกมีฝาเป็น “ภาชนะเก็บของ”; ช้อนโลหะและผ้าเช็ดมือเป็น “ไม่ใช่ภาชนะเก็บของ” เลือกคุณลักษณะหลักให้กฎของคุณ

กฎใดเหมาะสมที่สุดกับตัวอย่างฝึก? / Which rule best fits the training examples?

กิจกรรม 2: ชุดทดสอบและกรณีพิเศษ / Test set and edge cases

ใช้กฎเดิมกับของใหม่ที่ไม่อยู่ในตัวอย่างฝึก แล้วเลือกคำอธิบายที่รับผิดชอบที่สุด

ถุงผ้ามีเชือกรูดเก็บของได้ แต่ไม่ใช่กล่องหรือขวด กฎ “ต้องแข็งและมีฝา” จัดผิด เราควรทำอย่างไร?

แบบฝึกหัด / Quick check

1) คุณลักษณะ / feature คืออะไร?
2) ทำไมต้องแยกชุดทดสอบออกจากชุดฝึก?
3) เมื่อพบ edge case ควรทำอย่างไร?

สรุปและหลักฐาน / Exit ticket and evidence

เติมประโยค: “กฎของฉันใช้ไม่ได้กับ… เพราะ… ฉันจะตรวจต่อโดย…” / Complete: “My rule failed on… because… I will check next by…”

สำหรับผู้ใหญ่ / Adult note

ใช้บัตรคำหรือของจำลองที่สะอาดแทนการอัปโหลดภาพจริง ชวนเด็กอธิบายว่าทำไมเลือก feature นั้น แยกการ “ปรับกฎ” ออกจากการ “แอบแก้คำตอบชุดทดสอบ” และย้ำว่าระบบจำแนกไม่ควรใช้ตัดสินคุณค่า ความสามารถ พฤติกรรม หรือโอกาสของเด็ก
