CIT-02 · บทเรียน 11 / Lesson 11

ชุมชน กติกา และการตัดสินใจร่วมกัน
Community, Rules and Shared Decisions

คำถามนำ / Big question: เราจะสร้างกติกาและตัดสินใจร่วมกันอย่างไรให้ทุกคนมีเสียง ปลอดภัย และอยู่ร่วมกันได้?

ไปที่ / Jump to: เรียนรู้ · กติกา · การตัดสินใจ · แบบฝึกหัด

เป้าหมาย / Learning goals

  • อธิบายได้ว่ากติกาที่ดีช่วยคุ้มครองสิทธิ ความปลอดภัย และการมีส่วนร่วม
  • เปรียบเทียบการออกเสียงข้างมาก (vote) กับการหาฉันทามติ (consensus)
  • ใช้หลักฐาน รับฟังทุกคน และทบทวนผลกระทบก่อนตัดสินใจร่วมกัน

คำศัพท์ / Vocabulary

community — ชุมชน
กลุ่มคนที่อยู่ เรียน ทำงาน หรือดูแลพื้นที่ร่วมกัน
rule — กติกา
ข้อตกลงที่ช่วยให้คนอยู่ร่วมกันอย่างปลอดภัยและเป็นธรรม
vote — การออกเสียง
แต่ละคนเลือกทางเลือก แล้วนับเสียงตามวิธีที่ตกลงไว้
consensus — ฉันทามติ
ทุกคนช่วยปรับข้อเสนอจนสามารถยอมรับและร่วมปฏิบัติได้ แม้ไม่ใช่ตัวเลือกแรกของทุกคน
minority voice — เสียงส่วนน้อย
ความคิดเห็นของกลุ่มที่มีจำนวนน้อยกว่า ซึ่งยังต้องได้รับการรับฟังและคุ้มครอง

หลัก VOICE ก่อนตัดสินใจ / VOICE before deciding

  1. V — Views: รับฟังมุมมองที่แตกต่าง / Hear different views.
  2. O — Options: สร้างทางเลือกมากกว่าหนึ่ง / Create more than one option.
  3. I — Information: ใช้ข้อมูลและเหตุผล / Use information and reasons.
  4. C — Check impact: ตรวจผลต่อคนทุกกลุ่ม / Check impact on every group.
  5. E — Explain and evaluate: อธิบายผลและทบทวนภายหลัง / Explain and review later.

เด็กมีสิทธิแสดงความคิดเห็นในเรื่องที่กระทบตนเอง และผู้ใหญ่ควรให้น้ำหนักตามวัยและวุฒิภาวะ การมีส่วนร่วมต้องปลอดภัย สมัครใจ เข้าถึงได้ และไม่ทำให้เด็กต้องเปิดเผยเรื่องส่วนตัว

กิจกรรม 1: ตรวจคุณภาพกติกา / Rule-quality check

สถานการณ์จำลอง: ห้องเรียนต้องกำหนดกติกาการใช้มุมอ่านหนังสือร่วมกัน เลือกกติกาที่เป็นธรรมที่สุด

1) กติกาข้อใดชัดเจนและใช้กับทุกคน?
2) ถ้าผู้เรียนบางคนอ่านช้าหรือใช้เครื่องช่วย กติกาควรทำอย่างไร?
3) เมื่อกติกาไม่แก้ปัญหา ควรทำอย่างไร?

กิจกรรม 2: Vote หรือ consensus? / Choose a decision process

สถานการณ์จำลองเชียงราย: กลุ่มบ้านเรียนมีเวลา 40 นาทีสำหรับกิจกรรมชุมชน ทางเลือกคือสำรวจต้นไม้ ทำป้ายลดขยะ หรืออ่านเรื่องเล่าให้เด็กเล็ก ทุกตัวอย่างเป็นเรื่องสมมติและไม่ใช้ชื่อจริง

1) ต้องเลือกกิจกรรมเดียวภายในเวลาสั้น และทุกทางเลือกปลอดภัยพอ ๆ กัน วิธีใดเหมาะ?
2) ข้อเสนออาจทำให้ผู้ใช้รถเข็นเข้าร่วมไม่ได้ ควรทำอะไรก่อน?
3) หลังตัดสินใจแล้วควรทำอย่างไร?

แบบฝึกหัด / Quick check

1) กติกาที่เป็นธรรมควรมีลักษณะใด?
2) การ vote ที่รับผิดชอบควรทำอะไร?
3) Consensus หมายถึงอะไร?

ชิ้นงาน / Portfolio evidence

ออกแบบ “กติกาชุมชน 1 ข้อ” พร้อมเหตุผล กลุ่มที่ได้รับผลกระทบ วิธีรับฟังเสียงส่วนน้อย และวันที่ทบทวน โดยใช้ชื่อสมมติหรือรหัสผลงานเท่านั้น

เปิดใบงาน A4 / Open worksheets คู่มือผู้สอน / Teacher guide

ความปลอดภัยและความเป็นส่วนตัว / Safety and privacy

ไม่เปิดเผยเหตุการณ์ขัดแย้งจริง ชื่อจริง ที่อยู่ ภาพถ่าย หรือข้อมูลส่วนตัวในกิจกรรม หากประเด็นเกี่ยวข้องกับการกลั่นแกล้ง ความรุนแรง หรือความไม่ปลอดภัย ให้หยุดกิจกรรมและบอกผู้ใหญ่ที่ไว้ใจได้

การตัดสินใจร่วมกันที่ดีไม่ใช่เพียงการนับเสียง แต่คือการรับฟัง อธิบายเหตุผล คุ้มครองทุกคน และทบทวนผล
