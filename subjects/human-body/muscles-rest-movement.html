HB-03 · บทเรียน 3 / Lesson 3

กล้ามเนื้อ การพัก และการเคลื่อนไหวที่เหมาะกับวัย
Muscles, Rest and Age-Appropriate Movement

คำถามนำ / Big question: กล้ามเนื้อทำงานอย่างไร และเราจะวางแผนการเคลื่อนไหวที่สนุก ปลอดภัย และไม่แข่งขันกับร่างกายของคนอื่นได้อย่างไร?

เป้าหมาย / Learning goals

  • อธิบายว่ากล้ามเนื้อช่วยเคลื่อนไหว หายใจ สูบฉีดเลือด และย่อยอาหาร
  • สังเกตสัญญาณความหนักของกิจกรรมด้วย talk test แบบง่าย
  • วางแผนกิจกรรมให้หลากหลาย สนุก เหมาะกับวัย และมีช่วงพัก
  • สร้างกราฟกิจกรรมรายวันโดยไม่บันทึกชื่อ น้ำหนัก หรือข้อมูลสุขภาพส่วนบุคคล

คำศัพท์ / Vocabulary

muscle
กล้ามเนื้อ เนื้อเยื่อที่หดตัวเพื่อสร้างการเคลื่อนไหว
contract
หดตัว / become shorter and tighter
recover
ฟื้นตัวหรือพักให้พร้อมทำงานอีกครั้ง
moderate activity
กิจกรรมระดับปานกลาง หัวใจเต้นเร็วขึ้นแต่ยังพูดเป็นประโยคได้
evidence
หลักฐานจากสิ่งที่สังเกตหรือบันทึก

ข้อมูลสำคัญ / Key facts

กล้ามเนื้อไม่ได้มีหน้าที่เพียงยกของ กล้ามเนื้อช่วยให้เราเดิน หายใจ สูบฉีดเลือด และเคลื่อนอาหารผ่านระบบย่อยอาหาร กล้ามเนื้อมักทำงานร่วมกับกระดูกและข้อต่อ

เด็กวัยเรียนควรมีโอกาสเคลื่อนไหวรวมอย่างน้อยประมาณ 60 นาทีต่อวัน โดยเลือกกิจกรรมที่หลากหลาย เหมาะกับวัย สนุก และปรับได้ตามความสามารถ การเคลื่อนไหวสะสมเป็นช่วงสั้น ๆ ได้ ไม่จำเป็นต้องทำต่อเนื่องครั้งเดียว

ขอบเขตสุขภาพ: บทเรียนนี้ไม่ใช้วินิจฉัยโรค ไม่ใช้เปรียบเทียบรูปร่าง ความเร็ว หรือความแข็งแรงระหว่างเด็ก หากมีอาการเจ็บหน้าอก หายใจลำบากผิดปกติ หน้ามืด ปวดเฉียบพลัน หรืออาการต่อเนื่อง ให้หยุดและบอกผู้ใหญ่ทันที

ตัววัดความหนักแบบพูดคุย / Talk-test explorer

เลือกสถานการณ์จำลอง / Choose a fictional situation

สร้างแผนการเคลื่อนไหว / Build a movement plan

เลือกกิจกรรมจากแต่ละกลุ่ม ไม่ต้องแข่งขันหรือทำให้ครบทุกอย่างในครั้งเดียว

กิจกรรมที่สนุกและทำได้ / Enjoyable and possible activities
แผนพักและปรับ / Rest and adapt

กิจกรรมลงมือทำ / Observation activity

  1. เลือกกิจกรรมเบา ๆ ที่ปลอดภัย เช่น เดิน เต้น หรือยืดเหยียดแบบสบาย 3–5 นาที
  2. ก่อนและหลังทำ ให้บันทึกว่าพูดคุยได้ง่าย ปานกลาง หรือควรหยุดพัก
  3. เขียนสิ่งที่ช่วยให้รู้สึกพร้อม เช่น น้ำ พื้นที่ปลอดภัย รองเท้าเหมาะสม หรือผู้ใหญ่ดูแล
  4. ไม่จับเวลาแข่ง ไม่เปรียบเทียบจำนวนครั้ง และไม่ฝืนความเจ็บปวด

แบบประเมิน / Quick check

1) กล้ามเนื้อช่วยอะไรบ้าง?
2) แผนใดเหมาะกับวัยมากกว่า?
3) เมื่อเวียนหัวหรือเจ็บผิดปกติควรทำอย่างไร?
