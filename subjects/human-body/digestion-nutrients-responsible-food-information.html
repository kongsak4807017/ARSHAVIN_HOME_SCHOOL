ระบบย่อยอาหาร สารอาหาร และข้อมูลอาหารที่รับผิดชอบ
Digestion, Nutrients and Responsible Food Information

คำถามนำ / Big question: อาหารเดินทางผ่านร่างกายอย่างไร และเราจะอ่านข้อมูลอาหารโดยไม่ตัดสินคนหรือจำกัดอาหารอย่างไม่ปลอดภัยได้อย่างไร?

ไปที่ / Jump to: เรียนรู้ · เส้นทางการย่อย · อ่านข้อมูลอาหาร · แบบฝึกหัด

เป้าหมาย / Learning goals

  • เรียงเส้นทางอาหารจากปากไปยังลำไส้ใหญ่ และอธิบายการย่อยกับการดูดซึมอย่างง่าย
  • เชื่อมสารอาหารกับหน้าที่ เช่น พลังงาน การเจริญเติบโต การซ่อมแซม และการทำงานของร่างกาย
  • อ่านฉลากสมมติอย่างมีบริบท โดยไม่วินิจฉัยสุขภาพ ไม่จัดอันดับรูปร่าง และไม่สั่งตัดอาหาร

คำศัพท์ / Vocabulary

digestion — การย่อยอาหาร
การทำให้อาหารแตกเป็นส่วนเล็กพอที่ร่างกายดูดซึมและใช้ได้
peristalsis — การบีบตัวเป็นคลื่น
การหดและคลายของกล้ามเนื้อที่ช่วยดันอาหารไปตามทางเดินอาหาร
nutrient — สารอาหาร
สารจากอาหารและเครื่องดื่มที่ร่างกายใช้เพื่อพลังงาน การเจริญเติบโต การซ่อมแซม และการทำงานต่าง ๆ
absorption — การดูดซึม
การเคลื่อนสารอาหารที่ย่อยแล้วผ่านผนังลำไส้เข้าสู่เลือดหรือระบบน้ำเหลือง
serving — หน่วยบริโภค
ปริมาณอ้างอิงบนฉลาก ไม่ใช่คำสั่งว่าทุกคนต้องกินเท่ากัน

เส้นทางหลัก / The main path

  1. ปาก / Mouth: การเคี้ยวทำให้อาหารเล็กลง น้ำลายช่วยให้อาหารชื้นและเริ่มย่อยแป้งบางส่วน
  2. หลอดอาหาร / Esophagus: การบีบตัวเป็นคลื่นดันอาหารไปยังกระเพาะ ไม่ใช่แรงโน้มถ่วงเพียงอย่างเดียว
  3. กระเพาะอาหาร / Stomach: กล้ามเนื้อคลุกอาหารกับกรดและเอนไซม์
  4. ลำไส้เล็ก / Small intestine: ย่อยต่อและดูดซึมสารอาหารส่วนใหญ่เข้าสู่ร่างกาย
  5. ลำไส้ใหญ่ / Large intestine: ดูดซึมน้ำเพิ่มเติมและเปลี่ยนสิ่งที่เหลือเป็นอุจจาระ

อวัยวะช่วยย่อย / Helper organs: ตับสร้างน้ำดี ถุงน้ำดีเก็บน้ำดี และตับอ่อนสร้างน้ำย่อยที่ส่งเข้าสู่ลำไส้เล็ก

ขอบเขตสุขภาพ: บทเรียนนี้ไม่ขอให้บอกน้ำหนัก รูปร่าง อาหารที่กิน อาการ โรค ภูมิแพ้ หรือประวัติครอบครัว ไม่ใช้ฉลากตัดสินว่าอาหารหรือคน “ดี/แย่” และไม่แนะนำให้ตัดอาหารหรือเปลี่ยนแผนรักษา หากมีอาการ ปัญหาการกิน หรือข้อจำกัดทางการแพทย์ ให้บอกผู้ใหญ่ที่ไว้ใจได้และปรึกษาบุคลากรสุขภาพ

กิจกรรม 1: เรียงเส้นทางอาหาร / Build the digestion path

เลือกอวัยวะถัดไปตามลำดับ ปุ่มที่ถูกต้องจะเปิดขั้นต่อไป ไม่มีการจับเวลาและไม่ต้องลากวาง

เริ่มที่ปาก / Start at the mouth.

กิจกรรม 2: อ่านข้อมูลอาหารอย่างรับผิดชอบ / Read food information responsibly

ฉลากสมมติ: “ข้าวโอ๊ตกล้วยอบ” หนึ่งหน่วยบริโภคตามฉลาก มีคาร์โบไฮเดรต ใยอาหาร โปรตีน ไขมัน โซเดียม และน้ำตาลที่เติม ข้อมูลนี้สร้างขึ้นเพื่อฝึกอ่าน ไม่ใช่คำแนะนำส่วนบุคคล

1) ก่อนเปรียบเทียบตัวเลข ควรตรวจอะไร?
2) ข้อสรุปใดรับผิดชอบที่สุด?
3) เพื่อนบอกว่า “กินอาหารนี้แล้วผอมแน่นอน” ควรตอบอย่างไร?

แบบฝึกหัดท้ายบท / Quick check

1) สารอาหารส่วนใหญ่ถูกดูดซึมที่ใด?
2) การบีบตัวเป็นคลื่นทำหน้าที่อะไร?
3) วิธีใช้ข้อมูลอาหารที่ปลอดภัยกว่าคือข้อใด?
