HB-08 · บทเรียน 32 / Lesson 32

ระบบประสาท สมอง ประสาทสัมผัส และข้อมูลสุขภาพที่รับผิดชอบ
Nervous System, Brain, Senses and Responsible Health Information

คำถามนำ / Big question: ร่างกายรับสัญญาณ ตีความ และตอบสนองอย่างไร และเราจะใช้ข้อมูลเกี่ยวกับประสาทสัมผัสโดยไม่วินิจฉัยตนเองหรือเปรียบเทียบคนได้อย่างไร?

ไปที่ / Jump to: เรียนรู้ · เส้นทางสัญญาณ · ตัดสินใจอย่างปลอดภัย · แบบฝึกหัด

เป้าหมาย / Learning goals

  • อธิบายเส้นทาง สิ่งเร้า → ตัวรับความรู้สึก → เส้นประสาท → สมอง/ไขสันหลัง → การตอบสนอง
  • ยกตัวอย่างการมองเห็นและการได้ยินโดยแยกอวัยวะรับสัญญาณออกจากการตีความของสมอง
  • เลือกการปฏิบัติที่ลดอันตรายต่อประสาทสัมผัส โดยไม่ใช้การแข่งขันเวลาตอบสนองหรือข้อมูลเดียววินิจฉัยโรค

คำศัพท์ / Vocabulary

stimulus — สิ่งเร้า
การเปลี่ยนแปลงที่ตัวรับตรวจพบได้ เช่น แสง เสียง หรือการสัมผัส
receptor — ตัวรับความรู้สึก
เซลล์หรือส่วนของอวัยวะที่เปลี่ยนสิ่งเร้าเป็นสัญญาณประสาท
nerve — เส้นประสาท
ทางส่งสัญญาณระหว่างส่วนต่าง ๆ ของร่างกาย สมอง และไขสันหลัง
brain — สมอง
อวัยวะที่รับและประมวลผลข้อมูล ช่วยวางแผน เรียนรู้ จดจำ และควบคุมการตอบสนองหลายแบบ
spinal cord — ไขสันหลัง
ทางสื่อสารหลักระหว่างสมองกับร่างกาย และเกี่ยวข้องกับการตอบสนองอัตโนมัติบางชนิด
response — การตอบสนอง
การเปลี่ยนแปลงหรือการกระทำหลังระบบประสาทประมวลข้อมูล

แบบจำลองมีขอบเขต / Model limits

แบบจำลองนี้ย่อระบบที่ซับซ้อนมากให้เหลือเส้นทางหลัก สัญญาณหลายชนิดเดินทางพร้อมกัน สมองใช้ประสบการณ์ ความสนใจ และบริบทช่วยตีความ และการตอบสนองบางชนิดเกี่ยวข้องกับไขสันหลังด้วย

การมองเห็นเริ่มจากแสงผ่านส่วนต่าง ๆ ของตาไปยังจอประสาทตา ซึ่งเปลี่ยนแสงเป็นสัญญาณไฟฟ้า แล้วส่งผ่านเส้นประสาทตาไปยังสมอง การได้ยินเริ่มจากคลื่นเสียงทำให้ส่วนต่าง ๆ ของหูสั่นและเปลี่ยนเป็นสัญญาณที่เส้นประสาทหูส่งไปสมอง

ขอบเขตสุขภาพและความเป็นส่วนตัว: ไม่ต้องบอกอาการ ประวัติการมองเห็นหรือได้ยิน คะแนนการเรียน โรค ยา หรือข้อมูลครอบครัว บทเรียนนี้ไม่ทดสอบสายตา การได้ยิน สมาธิ ความจำ ออทิซึม ADHD หรือโรคระบบประสาท หากมีการมองเห็นหรือได้ยินเปลี่ยนฉับพลัน ปวดศีรษะรุนแรง เป็นลม ชัก อ่อนแรง สับสน บาดเจ็บศีรษะ หรือกังวลเรื่องอาการ ให้หยุดกิจกรรม บอกผู้ใหญ่ที่ไว้ใจได้ และรับคำแนะนำจากบุคลากรสุขภาพ

กิจกรรม 1: สร้างเส้นทางสัญญาณ / Build a signal path

เลือกส่วนตามลำดับ ใช้ปุ่มได้ด้วยคีย์บอร์ด ไม่มีการจับเวลาและไม่มีการจัดอันดับความเร็วของผู้เรียน

เริ่มที่สิ่งเร้า / Start with the stimulus.

กิจกรรม 2: ประสาทสัมผัสและการตัดสินใจที่ปลอดภัย / Sensory safety decisions

ทุกสถานการณ์เป็นเรื่องสมมติ ไม่มีการเก็บอาการหรือข้อมูลสุขภาพรายบุคคล

1) เสียงจากหูฟังสมมติดังจนรู้สึกไม่สบาย ควรทำอย่างไร?
2) เพื่อนสมมติมองกระดานไม่ชัด ควรใช้ข้อมูลอย่างไร?
3) เกมสมมติวัดเวลาตอบสนองครั้งเดียวแล้วบอกว่า “สมองผิดปกติแน่นอน” ควรตอบอย่างไร?

แบบฝึกหัดท้ายบท / Quick check

1) ลำดับพื้นฐานใดเหมาะสมที่สุด?
2) ข้อใดถูกต้องเกี่ยวกับการมองเห็น?
3) วิธีใช้ข้อมูลประสาทสัมผัสที่รับผิดชอบคือข้อใด?
