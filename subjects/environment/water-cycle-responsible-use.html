ENV-02 · บทเรียน 2 / Lesson 2

น้ำ วัฏจักรน้ำ และการใช้น้ำอย่างรับผิดชอบ
Water, the Water Cycle and Responsible Use

คำถามนำ / Big question: น้ำหยดหนึ่งเดินทางผ่านอากาศ ดิน แม่น้ำ บ้าน และชุมชนได้อย่างไร และเราจะใช้น้ำโดยไม่สร้างปัญหาให้คนปลายน้ำได้อย่างไร?

ไปที่ / Jump to: เรียนรู้ · วัฏจักรน้ำ · ลุ่มน้ำเชียงราย · แบบฝึกหัด · สำหรับผู้ใหญ่

เป้าหมาย / Learning goals

  • เรียงลำดับกระบวนการสำคัญของวัฏจักรน้ำได้: evaporation, condensation, precipitation, collection และ infiltration
  • อธิบายได้ว่าน้ำในธรรมชาติหมุนเวียน แต่ปริมาณน้ำสะอาดที่ใช้ได้ในแต่ละเวลาและสถานที่มีจำกัด
  • เลือกวิธีใช้น้ำที่ลดการสูญเปล่า ลดมลพิษ และคำนึงถึงคนในลุ่มน้ำเดียวกัน

คำศัพท์ / Vocabulary

evaporation — การระเหย
น้ำของเหลวได้รับพลังงานความร้อนและเปลี่ยนเป็นไอน้ำ
condensation — การควบแน่น
ไอน้ำเย็นลงและรวมตัวเป็นหยดน้ำเล็ก ๆ เช่น เมฆ
precipitation — หยาดน้ำฟ้า
น้ำตกจากเมฆ เช่น ฝน ลูกเห็บ หรือหิมะ
infiltration — การซึมลงดิน
น้ำไหลเข้าสู่ช่องว่างในดินและอาจเติมน้ำใต้ดิน
watershed — ลุ่มน้ำ
พื้นที่ที่น้ำไหลรวมไปยังแม่น้ำ ลำห้วย หรือแหล่งน้ำเดียวกัน

หลักสำคัญ / Key ideas

ดวงอาทิตย์ให้พลังงานแก่น้ำ น้ำระเหยขึ้นสู่บรรยากาศ เย็นลงเป็นเมฆ ตกกลับมาเป็นฝน แล้วไหลบนผิวดินหรือซึมลงดิน ก่อนจะกลับสู่แม่น้ำ ทะเลสาบ หรือทะเล วัฏจักรนี้ไม่มีจุดเริ่มต้นตายตัวและเกิดหลายกระบวนการพร้อมกันได้

แม้น้ำหมุนเวียนอยู่เสมอ แต่น้ำสะอาดที่เข้าถึงได้อาจไม่พอในฤดูแล้ง อาจปนเปื้อนจากขยะ น้ำมัน สารเคมี หรือดินตะกอน และอาจกระจายไม่เท่ากันระหว่างต้นน้ำ–กลางน้ำ–ปลายน้ำ

หลัก CARE ก่อนใช้น้ำ / CARE before using water

  1. CHECK — ตรวจ: จำเป็นต้องใช้น้ำเท่าไร?
  2. AVOID — หลีกเลี่ยง: ปิดก๊อกและไม่ปล่อยน้ำไหลทิ้ง
  3. REUSE — ใช้ซ้ำอย่างเหมาะสม: ใช้น้ำล้างผักรดต้นไม้เมื่อปลอดภัย
  4. EXCLUDE POLLUTION — ไม่เติมมลพิษ: ไม่เทน้ำมัน สี ยา หรือสารเคมีลงท่อหรือแหล่งน้ำ

บทเรียนนี้เป็นการศึกษาทั่วไป ไม่ยืนยันว่าตัวอย่างน้ำใดปลอดภัยสำหรับดื่ม น้ำที่ไม่ทราบแหล่งหรือคุณภาพต้องให้ผู้ใหญ่และหน่วยงานที่เกี่ยวข้องตรวจสอบ

กิจกรรม 1: นักเดินทางหยดน้ำ / Water-drop journey

เลือกกระบวนการเพื่อพาหยดน้ำจากแต่ละสถานีไปยังสถานีถัดไป ใช้ปุ่มและตัวเลือกมาตรฐาน จึงทำได้ด้วยคีย์บอร์ดโดยไม่ต้องลากวาง

กระบวนการถัดไปคืออะไร? / What happens next?

กิจกรรม 2: การตัดสินใจในลุ่มน้ำเชียงราย / Chiang Rai watershed decisions

สถานการณ์จำลอง: บ้าน โรงเรียน สวน และชุมชนอยู่ในลุ่มน้ำเดียวกัน การกระทำต้นน้ำอาจมีผลต่อคนกลางน้ำและปลายน้ำ เลือกคำตอบที่รับผิดชอบที่สุดในแต่ละข้อ

1) หลังล้างผัก มีน้ำสะอาดที่ไม่ผสมน้ำยาหนึ่งกะละมัง
2) พบคราบน้ำมันใกล้ท่อระบายน้ำ
3) โรงเรียนต้องรดต้นไม้ในช่วงแล้ง

แบบฝึกหัด / Quick check

1) เมื่อไอน้ำเย็นลงและรวมเป็นหยดเล็ก ๆ เรียกว่าอะไร?
2) ข้อใดอธิบายลุ่มน้ำได้ดีที่สุด?
3) วิธีใดช่วยทั้งประหยัดน้ำและลดมลพิษ?

ชิ้นงาน / Portfolio evidence

วาดแผนที่ย่อของบ้านหรือโรงเรียน แสดงจุดใช้น้ำ จุดที่น้ำไหลออก และหนึ่งวิธีลดการสูญเปล่าหรือมลพิษ โดยไม่ต้องระบุที่อยู่จริง

เปิดใบงาน A4 / Open A4 worksheets คู่มือผู้สอน / Teacher guide

สำหรับผู้ใหญ่ / Adult note

ใช้ภาชนะสะอาดและน้ำประปาหรือน้ำที่ผู้ใหญ่ยืนยันว่าปลอดภัยสำหรับกิจกรรม ห้ามชิมน้ำจากแหล่งทดลอง ห้ามสัมผัสน้ำเสีย คราบน้ำมัน สารเคมี หรือภาชนะไม่ทราบชนิด ให้เด็กแจ้งผู้ใหญ่ทันทีเมื่อพบความเสี่ยง

ตัวอย่างเชียงรายเป็นสถานการณ์เรียนรู้ทั่วไป ไม่ใช่รายงานสถานการณ์น้ำปัจจุบันหรือคำรับรองคุณภาพน้ำเฉพาะพื้นที่

น้ำเชื่อมคนทั้งลุ่มน้ำ การใช้น้ำอย่างรับผิดชอบเริ่มจากการลดการสูญเปล่าและไม่ส่งมลพิษต่อไปยังผู้อื่น
