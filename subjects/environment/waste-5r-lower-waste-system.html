ขยะ หลัก 5R และการออกแบบระบบลดของเสีย
Waste, the 5Rs and Designing a Lower-Waste System

คำถามนำ / Big question: เราจะลดขยะตั้งแต่ก่อนเกิด แทนที่จะพึ่งการรีไซเคิลเพียงอย่างเดียวได้อย่างไร?

ไปที่ / Jump to: เรียนรู้ · สำรวจขยะสมมติ · ออกแบบระบบ · แบบฝึกหัด · สำหรับผู้ใหญ่

เป้าหมาย / Learning goals

  • อธิบายว่า waste prevention หรือการป้องกันขยะสำคัญกว่าการจัดการเมื่อขยะเกิดแล้ว
  • ใช้หลัก Refuse–Reduce–Reuse–Repair–Recycle ตามลำดับก่อนหลัง
  • ออกแบบระบบลดของเสียที่ปลอดภัย วัดผลได้ และเหมาะกับบ้านหรือชุมชน

คำศัพท์ / Vocabulary

waste — ของเสียหรือขยะ
วัสดุที่ผู้ใช้ไม่ต้องการแล้ว แต่บางชิ้นยังป้องกัน ใช้ซ้ำ ซ่อม หรือรีไซเคิลได้
source reduction — การลดที่ต้นทาง
ป้องกันไม่ให้ของเสียเกิดขึ้นตั้งแต่แรก
reuse — ใช้ซ้ำ
ใช้สิ่งของเดิมอีกครั้งโดยยังคงหน้าที่เดิมหรือปรับใช้ใหม่อย่างปลอดภัย
repair — ซ่อม
แก้ไขให้สิ่งของกลับมาใช้งานได้ โดยผู้ใหญ่หรือช่างที่เหมาะสมเมื่อมีเครื่องมือ ไฟฟ้า หรือความเสี่ยง
recycle — รีไซเคิล
แยกและส่งวัสดุเข้าสู่ระบบแปรรูปตามกติกาท้องถิ่น ไม่ใช่วัสดุทุกชนิดจะรับรีไซเคิลเหมือนกัน

หลัก 5R ตามลำดับ / The 5Rs in order

  1. Refuse — ปฏิเสธ: ไม่รับสิ่งที่ไม่จำเป็น เช่น ช้อนส้อมใช้ครั้งเดียวเมื่อมีของใช้ส่วนตัวที่สะอาด
  2. Reduce — ลด: ใช้เท่าที่จำเป็น เลือกของทนทานหรือบรรจุภัณฑ์น้อยลง
  3. Reuse — ใช้ซ้ำ: ใช้ถุง กล่อง หรือกระดาษอีกครั้งเมื่อสะอาดและเหมาะสม
  4. Repair — ซ่อม: ซ่อมเสื้อผ้าหรือสิ่งของง่าย ๆ โดยมีผู้ใหญ่กำกับ ห้ามเด็กซ่อมเครื่องใช้ไฟฟ้า แบตเตอรี่ หรือของมีคมเอง
  5. Recycle — รีไซเคิล: แยกวัสดุที่ระบบท้องถิ่นรับหลังจากลด ใช้ซ้ำ หรือซ่อมไม่ได้แล้ว

ข้อจำกัด / Limit: การรีไซเคิลไม่ใช่คำตอบแรก วัสดุเปื้อนอาหาร สารเคมี แบตเตอรี่ หลอดไฟ อุปกรณ์อิเล็กทรอนิกส์ และของมีคมต้องให้ผู้ใหญ่จัดการตามระบบเฉพาะ

กิจกรรม 1: สำรวจขยะสมมติ / Fictional waste audit

ไม่มีการสัมผัสขยะจริง เลือกการจัดการที่เหมาะที่สุดสำหรับบัตรสถานการณ์สมมติแต่ละใบ

1) ร้านเสนอหลอดพลาสติก แต่เรามีแก้วพร้อมดื่มได้
2) สมุดเหลือกระดาษสะอาดหลายหน้า
3) เสื้อกระดุมหลุด แต่ผ้ายังดี
4) พบถ่านไฟฉายใช้แล้วในกล่อง

กิจกรรม 2: ออกแบบระบบลดของเสีย / Design a lower-waste system

สถานการณ์สมมติ: มุมอาหารว่างของบ้านเรียนมีแก้วใช้ครั้งเดียว 20 ใบต่อสัปดาห์ เลือกระบบที่ลดของเสียตั้งแต่ต้นทางและตรวจสอบผลได้

การเปลี่ยนระบบ / System change
หลักฐานที่ควรเก็บ / Evidence

แบบฝึกหัด / Quick check

1) เหตุใด Refuse และ Reduce จึงมาก่อน Recycle?
2) เมื่อพบของมีคมหรือภาชนะสารเคมีควรทำอย่างไร?
3) หลักฐานใดบอกได้ดีที่สุดว่าระบบลดขยะได้จริง?

สำหรับผู้ใหญ่ / Adult guidance

บทเรียนนี้ใช้บัตรสถานการณ์และของสะอาดเท่านั้น ไม่ให้เด็กค้น ถ่ายรูป ชั่ง หรือสัมผัสขยะจริง ห้ามสัมผัสของมีคม แก้วแตก เข็ม แบตเตอรี่รั่ว หลอดไฟ สารเคมี ยา ขยะสุขอนามัย ซากสัตว์ หรือวัสดุไม่ทราบชนิด ให้กั้นพื้นที่และให้ผู้ใหญ่ผู้รับผิดชอบจัดการ
