ENV-01 · บทเรียน 3 / Lesson 3

อ่านอากาศ เลือกทางปลอดภัยกว่า
Read the Air, Choose a Safer Action

คำถามนำ / Big question: เมื่อเห็นค่าฝุ่นหรือสีคุณภาพอากาศ เราจะตรวจข้อมูลและปรับกิจกรรมอย่างไรโดยไม่ตื่นตระหนก?

ไปที่ / Jump to: เรียนรู้ · อ่านข้อมูล · วางแผน · แบบฝึกหัด · สำหรับผู้ใหญ่

เป้าหมาย / Learning goals

  • อธิบายได้ว่า PM2.5 คืออนุภาคขนาดเล็กมากในอากาศ ไม่ใช่ชื่อของควันชนิดเดียว
  • อ่าน ค่า เวลา สถานที่ สี และแหล่งข้อมูล ก่อนตัดสินใจ
  • เลือกการกระทำที่ลดการสัมผัสฝุ่นโดยขอความช่วยเหลือจากผู้ใหญ่เมื่อจำเป็น

คำศัพท์ / Vocabulary

PM2.5 — ฝุ่นละอองขนาดเล็ก
อนุภาคในอากาศที่มีขนาดไม่เกิน 2.5 ไมโครเมตร จึงเล็กจนมองไม่เห็นทีละอนุภาค
air-quality reading — ค่าคุณภาพอากาศ
ข้อมูลจากสถานีหรือเครื่องวัด ซึ่งต้องอ่านพร้อมหน่วย เวลา และสถานที่
exposure — การสัมผัส
การอยู่ในอากาศที่มีมลพิษ โดยความเสี่ยงขึ้นกับระดับ ระยะเวลา กิจกรรม และสุขภาพของแต่ละคน
sensitive group — กลุ่มไวต่อมลพิษ
เช่น เด็ก ผู้สูงอายุ และผู้มีโรคหัวใจหรือปอด ซึ่งอาจได้รับผลกระทบง่ายกว่า

หลักสำคัญ / Key ideas

PM2.5 เล็กประมาณหนึ่งในสามสิบของความกว้างเส้นผมมนุษย์โดยเฉลี่ย และสามารถเข้าสู่ปอดได้ลึก เด็กเป็นกลุ่มที่ควรระวังเพราะปอดและร่างกายยังเติบโต อย่างไรก็ตาม ค่าหนึ่งค่าไม่ใช่คำวินิจฉัยโรค และสีจากคนละระบบอาจใช้ช่วงตัวเลขต่างกัน

อ่าน 5 อย่างก่อนเลือก / Read five things first

  1. WHAT — อะไร: เป็น PM2.5, AQI หรือข้อมูลชนิดอื่น?
  2. WHERE — ที่ไหน: สถานีอยู่ใกล้พื้นที่ของเราหรือไม่?
  3. WHEN — เมื่อไร: ข้อมูลอัปเดตล่าสุดตอนไหน?
  4. SOURCE — จากไหน: เป็นหน่วยงานหรือเครื่องวัดที่อธิบายวิธีวัดได้หรือไม่?
  5. ACTION — ทำอะไร: ปรับเวลา ความหนัก หรือสถานที่ของกิจกรรม และบอกผู้ใหญ่เมื่อมีอาการ

บทเรียนนี้ใช้ข้อมูลจำลองเพื่อฝึกอ่าน ไม่ใช่ประกาศคุณภาพอากาศปัจจุบัน และไม่ใช้แทนคำแนะนำแพทย์หรือประกาศท้องถิ่น

กิจกรรม 1: นักอ่านอากาศ / Air-data reader

เลื่อนแถบเพื่อดูสถานการณ์จำลอง แล้วเลือกการตีความที่ดีที่สุด สีเป็นภาษาง่ายสำหรับบทเรียน ไม่ใช่มาตรฐานทางการใหม่

ข้อใดเป็นการอ่านข้อมูลที่รอบคอบที่สุด?

กิจกรรม 2: เลือกแผนที่ปลอดภัยกว่า / Safer-action planner

สถานการณ์จำลอง: เวลา 15:00 น. ค่าใกล้โรงเรียนอยู่ในระดับที่ควรลดกิจกรรมหนักกลางแจ้ง เพื่อนชวนวิ่งแข่ง 45 นาที คุณจะเลือกอะไร?

เลือกแผนที่เหมาะสมที่สุด

แบบฝึกหัด / Quick check

1) ก่อนใช้ค่าฝุ่น เราควรตรวจอะไร?
2) เหตุใดเด็กจึงควรระวังมลพิษทางอากาศ?
3) เมื่อไอ หายใจลำบาก แน่นหน้าอก หรือรู้สึกผิดปกติระหว่างวันที่อากาศไม่ดี ควรทำอย่างไร?

หลักฐานและบทบาทผู้ใหญ่ / Evidence and adult support

พิมพ์ใบงาน 2 หน้าเพื่อบันทึกข้อมูลจำลอง เปรียบเทียบแผนกิจกรรม และสะท้อนเหตุผล ผู้ใหญ่ช่วยตรวจข้อมูลจริงจากหน่วยงานที่รับผิดชอบในพื้นที่ ไม่ควรให้เด็กตัดสินใจเรื่องหยุดเรียน หน้ากาก ยา หรืออาการเจ็บป่วยตามลำพัง
